คู่มือการเดินทางไปยังจุดนัดหมาย IT Square หลักสี่ ชั้น 3

กรณีเดินทางมาก่อนเวลาห้างเปิด (ก่อน 10:00 น.)

หากคุณมีนัดหมายก่อนเวลา 10:00 น. ซึ่งประตูหลักของห้างยังไม่เปิด ให้ใช้เส้นทางพิเศษผ่านลานจอดรถตามขั้นตอนดังนี้ครับ

📍 ข้อมูลสถานที่

สถานที่: IT Square หลักสี่ ชั้น 3 (บริเวณหน้าสำนักงานจัดหางาน)

พิกัด Google Maps: คลิกเพื่อดูแผนที่

🚶 ขั้นตอนการเดินทาง (Step-by-Step)

ขั้นตอนที่ 1: จุดเริ่มต้นหน้าธนาคารกรุงเทพ

เมื่อถึงหน้าห้าง IT Square ให้มองหา ธนาคารกรุงเทพ (Bangkok Bank Express) จากนั้นเดินตรงไปตามลูกศรสีขาว

ขั้นตอนที่ 2: ลงบันไดเลื่อน

เดินมาจะเจอบันไดเลื่อน ให้ลงบันไดเลื่อนเพื่อลงไปยังชั้นล่าง

ขั้นตอนที่ 3: เข้าสู่โซนลานจอดรถ

เมื่อลงบันไดเลื่อนมาแล้ว ให้เดินเลี้ยวขวาเพื่อออกไปยังพื้นที่ลานจอดรถ

ขั้นตอนที่ 4: ไปยังทางเข้าลิฟต์ (T2)

เดินตรงต่อมาในลานจอดรถ จะเห็นประตูทางเข้ากระจกที่มีป้าย PLAZA ENTRANCE (T2) ให้เดินเข้าไปด้านในเพื่อใช้ลิฟต์

ขั้นตอนที่ 5: ขึ้นลิฟต์ไปชั้น 3

ภายในลิฟต์ ให้กด ชั้น 3 (สังเกตจะมีป้ายกำกับว่า K COUNTER / Short Term Visa)

ขั้นตอนที่ 6: ถึงจุดนัดหมาย

เมื่อออกจากลิฟต์แล้ว ให้ เลี้ยวขวา แล้วเดินตรงไปเรื่อย ๆ จะพบกับจุดนัดหมาย สำนักงานจัดหางานกรุงเทพมหานครพื้นที่ 9 (โซนสีชมพู)

หมายเหตุ: หากเดินทางมาหลัง 10:00 น. สามารถใช้ลิฟต์หรือบันไดเลื่อนหลักภายในห้างขึ้นมายังชั้น 3 ได้ตามปกติ
